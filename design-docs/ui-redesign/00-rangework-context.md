# Rangework Context

Rangework is an Android app for golfers who want to practice with purpose. Instead of showing up at the range and hitting balls aimlessly, Rangework lets you plan your practice in advance — building structured drills and composing them into repeatable session templates you can follow on the day.

Target users:

- Amateur golfers
- Handicap 10-35
- Practice mostly alone
- Often plan practice before arriving at the range

Primary user goals:

- Create practice units
- Combine units into sessions
- Understand ball counts
- Follow a structured practice plan

Constraints:

- Material 3 must remain
- Existing color palette must remain
- Existing typography primitives must remain
- Android-native patterns only

Review priorities:

- UX
- Information architecture
- Discoverability
- Cognitive load
- Task completion speed
- Visual hierarchy

Avoid:

- Generic visual redesigns
- Dribbble-style concepts
- iOS patterns
