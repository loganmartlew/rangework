# Rangwork - Product Website

You are an expert web-developer skilled at implementing designs from a designer.

## Task Details

In `design-docs/RWK11-session-execution/design-spec` is all the details for a website design. It includes everything you will need to implement the website. It also contains a skill. Use the skill.

The website already exists as an Astro project in `apps/site`. You are to implement the new design into the existing site.

Use the same tools that are already available (tailwind, material icons, cva if needed).

The correct fonts and colors are already provided to the site through the `packages/ui-tokens` package. Any changes to that sort of thing should be in there, but there shouldn't be any changes.

The site has some existing assets and components (button, textlogo, svgs for logo and a mockup of part of the app). You can choose to use these, or discard them.

Light and dark mode are both in scope

Use UI/UX and accessibility best practives for this site. Implement animations where logical.

## Process

The first step is to generate a list of requirements. Work with the user, asking questions wherever needed, to produce a requirements document. End the current coding session afterwards. Persist the requirements to a markdown doc `design-docs/RWK11-session-execution/requirements.md`.

Next step is to create an implementation plan. This should contain all the technical details of how we will implement the design. This should be pretty detailed. Work with the user, asking questions wherever needed, to produce a document with the plan. End the current coding session afterwards. Persist the plan to a markdown doc `design-docs/RWK11-session-execution/implementation-plan.md`.

Finally, the user will prompt to implement the website. At this stage you may begin writing the code.
