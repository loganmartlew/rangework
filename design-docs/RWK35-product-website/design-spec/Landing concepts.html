The reframe

You've got the tools. You're missing the plan.

A launch monitor. A swing app. A GPS watch. A stat tracker. And still, when you show up at the range, you're standing over a pile of balls trying to remember what your coach said.

The gap is structure. Rangework fills it.

02 / 06  ·  The system

Build drills. Plan sessions. Execute on the range.

Step 01 · Build
Design your drills

Create reusable practice units with step-by-step instructions, ball counts, club assignments, notes, and a mental focus cue. "50-yard pitch shots: three landing zones, ten balls each."

Step 02 · Plan
Compose your session

Combine drills into a complete practice template. Set repeat counts, override clubs per slot, and watch the ball count total update in real time.

Step 03 · Execute
Run it on the range

Hit Start and Rangework turns the session into a guided, step-by-step flow. Tap to complete, auto-advance to the next. Walk away, come back, pick up exactly where you were.

Concept 03 / 05

The Spec Sheet

Direction
Page reads like a player's yardage book crossed with an engineering spec sheet. Two-column ruled layout, hairline tables, mono-heavy labels, document footer.
Tone
Serious, precise, almost reverent of the craft. Speaks to the golfer who keeps drills in a notebook because they actually use them.
Colour
Warm graphite and ink. Deep Fairway used as a single ink colour for table lines, mono values, and the live status dot. No fills, no gradients.
Type
DM Mono is dominant — section labels, table values, callouts. DM Sans reserved for the title statement and body prose only.
Structure
Spec header → title → PROBLEM/RESOLUTION table → SYSTEM (3-level diagram) → FEATURE INDEX (table) → REFERENCE SESSION (fully tabulated) → INSTALL block.

Practice with purpose.

A practice planning app that turns aimless ball-hitting into structured, repeatable sessions you can build at home and execute at the range. The range is where scores are made — this is the tool you've been doing it without.

Concept 05 / 05

Empty State

Direction
Extreme minimalism. Echoes the app's own empty/onboarding states. The brand mark used at oversized scale. One statement per screen-height, generous breathing room, almost nothing else.
Tone
Quiet, confident, almost monastic. Says less and trusts the reader more.
Colour
Pure warm graphite (#FAFAF8) throughout. Brand-rod green (#386044) on the mark and on numeric values only. Body text in muted on-surface.
Type
DM Sans 300 weight at very large display sizes (96–128px). Body paragraphs intentionally short. DM Mono only on the few numbers that earn it.
Structure
Six near-empty panels stacked centred: mark hero · one-sentence reframe · build/plan/execute triptych · features bulleted · sample session as a quote · CTA on a near-empty surface.
